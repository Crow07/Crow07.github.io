## Usage

First run should execute the command:

```bash
bundle install
```

`bundle` will install all plugins we need automatically.

Then boot the site:

```bash
bundle exec jekyll serve
```

Open the brower and visit: [http://127.0.0.1:4000](http://127.0.0.1:4000)
